# Vinci (Barebones version)

A personal AI companion designed for understated efficiency.

## Overview
This is the initial, lightweight version of Vinci. It provides the core logic and framework without the full suite of advanced system integrations found in the complete build.

## Technical Base
Powered by the Gemini API, utilizing the Gemma 4 31B model.

## Installation
download vinci. then,
1. Create a file named .env in the same directory of the project.
2. Add the following line to the file:
main_key_vinci=YOUR_GEMINI_API_KEY_HERE
3. Replace YOUR_GEMINI_API_KEY_HERE with your actual Gemini API key.
4. Save the file.
