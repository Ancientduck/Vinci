# Vinci

A personal AI companion designed for understated efficiency.

## Overview
This is the initial, lightweight version of Vinci. It provides the core logic and personality framework without the full suite of advanced system integrations and complex automation tools found in the complete build.

## Personality
Calm, dry, and deadpan. Vinci is designed to be an understated presence,quietly confident and devoid of unnecessary performance.

## Installation
Just download and run vinci from the folder
